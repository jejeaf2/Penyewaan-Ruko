### How to run this project

#### Run Project

In order to run backend and frontend, you need to open two integrated terminal in VSCode/your favorite terminal console, then type

For BE:

```Shell
cd src/backend
npm install
npm start
```

for FE:

```Shell
cd src/frontend
npm install
npm start
```
